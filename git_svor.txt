#Git spurningar og svör

1. Hvað gera eftirfarandi Linux skipanir?
	cd									breytir um hvaða möppu þú ert í
	cd ~								færir þig í heimamöppuna
	ls									birtir lista yfir skrár í möppu
	pwd									sýnir hvaða möppu þú ert í og leiðina að henni
	mkdir								býr til nýja möppu

2. Hvað gera eftirfarandi git skipanir og hvers vegna eru þær gagnlegar?
	git clone							klónar geymslu á tölvuna
	git log								sýnir öll commit
	git status							sýnir stöðu greinarinnar sem þú ert notandi
	git diff							sýnir breytingar í hverju commiti
	git checkout						breytir um grein

3. Hver er munurinn á eftirfarandi git skipunum:
	a) git diff 						sýnir breytingar í hverju commiti
	b) git diff --staged 				sýnir breytingar frá commiti sem þú velur og því sem þú ert að fara að commita
	c) git diff commit1 commit2			sýnir breytingar á milli tveggja commita
