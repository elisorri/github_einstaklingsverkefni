#Git spurningar og svör

1. Hvað gera eftirfarandi Linux skipanir?
	cd									breytir um hvaða möppu þú ert í
	cd ~								færir þig í heimamöppuna
	ls									birtir lista yfir skrár í möppu
	pwd									sýnir hvaða möppu þú ert í og leiðina að henni
	mkdir								býr til nýja möppu

2. Hvað gera eftirfarandi git skipanir og hvers vegna eru þær gagnlegar?
	git clone							klónar geymslu á tölvuna
	git log								sýnir öll commit
	git status							sýnir stöðu greinarinnar sem þú ert að nota
	git diff							sýnir breytingar í hverju commiti
	git checkout						breytir um grein

3. Hver er munurinn á eftirfarandi git skipunum:
	a) git diff 						sýnir breytingar í hverju commiti
	b) git diff --staged 				sýnir breytingar frá commiti sem þú velur og því sem þú ert að fara að commita
	c) git diff commit1 commit2			sýnir breytingar á milli tveggja commita

4. 	Hvað er útgáfustýring (e.version control)? 		útgáfustýring er að hafa skipurlagða stýringu á útgáfum forrita

5.	Hverjir eru helstu kostir við að nota GIT?		það gerir það auðveldara að vinna saman

6.	Hversu oft telur þú að eigi að gera commit í verkefni, rökstuddu?		fer eftir hversu stórt verkefnið er

7.	Hvað er "Working directory"?" "Staging area"?" og "Repository" í GIT? 		working directory er mappan sem er verið að vinna í. staging area er staðurinn þar sem þú færð að ráða hvað breytist þegar þú commitar. repository er staðurinn sem þú geymir verkefnið á
